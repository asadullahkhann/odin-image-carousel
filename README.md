# Image Carousel

A responsive image carousel that displays 12 SVG images. It automatically moves to the next image every 7 seconds, while also allowing manual navigation through buttons and dot indicators.

## Features

* Displays 12 SVG images
* Auto-slides every 7 seconds
* Next and previous navigation buttons
* Dot navigation for direct slide selection
* Smooth horizontal scale animation on next/previous transitions

## Built With

* HTML
* CSS
* JavaScript

## Usage

1. Carousel starts automatically on load.
2. Use **next/previous buttons** to navigate manually.
3. Click **dots** to jump to a specific slide.
4. Let it autoplay or control it manually.
